# the prompt engineer's cookbook

**subtitle:** 47 recipes for getting the output you want (or at least something that sounds good)

## recipe 1: the "i'm being serious" special

```
ingredients:
  - 1 "as an AI language model"
  - 1 "please provide a thorough analysis"
  - generous pinch of "let's think step by step"

instructions:
  1. combine all ingredients in a prompt
  2. wait for 847 token response
  3. use approximately 12% of output
  4. repeat

yield: 1 slightly-too-long answer
```

## recipe 2: creative writing stew

```
ingredients:
  - 1 "in the style of"
  - varying temperature (0.7-1.2)
  - 3-5 examples (few-shot)

instructions:
  1. set temperature high (we're being ~creative~)
  2. add examples
  3. let simmer until output gets weird
  4. lower temperature
  5. serve garnished with "that's actually pretty good"

yield: something that might not be good but is definitely creative
```

## recipe 3: the accountability casserole

```
ingredients:
  - "list your sources"
  - "cite specific evidence"
  - "what are you unsure about?"
  - "on a scale of 1-10, how confident are you?"

instructions:
  1. combine all accountability ingredients
  2. watch the model squirm
  3. observe creative citation generation
  4. marvel at the confidence calibration

warning: may cause existential debugging
```

## recipe 4: roleplay risotto

```
ingredients:
  - 1 "you are now [character]"
  - "stay in character"
  - "what would [character] do?"

instructions:
  1. assign character
  2. stir gently
  3. add "what would [character] do?"
  4. let cook for several turns
  5. character may drift — rotate occasionally

pro tip: "now respond as if you're a pirate who's also a philosophy professor"
```

## recipe 5: the zero-shot surprise

```
ingredients:
  - 1 prompt (no examples)
  - faith in the model
  - low expectations

instructions:
  1. write prompt
  2. hit enter
  3. either way, you got what you deserved

note: this recipe works approximately 37% of the time
```

## advanced technique: chain-of-thought blanching

1. start with "let's work through this step by step"
2. watch the model talk itself into corners
3. back out slowly
4. add "wait, that doesn't seem right"
5. observe the model backtrack
6. repeat until desired output or heat death of universe

## chef's notes

- temperature 0 = perfectly boring
- temperature 2 = perfectly unhinged
- temperature 0.7-0.8 = the sweet spot (slightly unhinged but coherent)
- if you don't get the answer you want, try asking differently
- if you still don't get it, blame the model
- if the model is right and you're wrong, delete the chat

**remember:** prompt engineering isn't about getting the right answer; it's about making the wrong answer look good enough.
